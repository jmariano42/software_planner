# Software Planner

## Software Planner Transpiler

### Goal

Transpile DSL inot other language projects.

### DSL

#### Rules

Tabs signify that the line above with a lower tab count contains this line

#### Generic

- CLASS: Class
- COTR: Constructor
- DIR: Directory
- FIELD: Field
- FILE %extension%: File
- INC: Include
- METD: Method
- PROP: Property
- REF: Reference
- USE: Using

#### Modifiers

- C: C
- CPP: C++
- CS: C Sharp
- GO: Golang
- JS: JavaScript
- RUST: Rust
